# GM Economy 测试清单

## 测试环境要求

- `features.enable_gm_economy = true`
- 测试账号有 GM 权限（`CUser::IsGmMode() = true`）
- 频道重启后生效

## 已迁移命令

| 命令 | 功能 | API | 上限 |
|------|------|-----|------|
| `//czdq<X>` | 充值点券 | `user:ChargeCera(X)` | `max_cera` (100000) |
| `//czdb<X>` | 充值代币 | `user:ChargeCeraPoint(X)` | `max_point` (100000) |
| `//czsd<X>` | 充值胜点 | `user:GainWinPoint(X)` | `max_win_point` (10000) |

## 暂缓命令（旧实现依赖 SQL）

- `//czsp` — SP（skill.remain_sp）
- `//cztp` — TP（skill.remain_sfp_1st）
- `//czqp` — QP（charac_quest_shop.qp）

执行上述命令时会提示"暂未迁移"。
