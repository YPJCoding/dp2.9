# GM Send Item 测试清单

## 测试环境要求

- `features.enable_gm_send_item = true`
- 测试账号有 GM 权限（`CUser::IsGmMode() = true`）
- 频道重启后生效

## 功能测试

### 1. 默认关闭时不执行

| # | 操作 | 预期结果 |
|---|------|----------|
| 1.1 | `features.enable_gm_send_item = false`，GM 输入 `//send100` | 命令透传，不发物品 |

### 2. 非 GM 使用时拒绝

| # | 操作 | 预期结果 |
|---|------|----------|
| 2.1 | 非 GM 账号输入 `//send100` | 提示"没有权限"，不发物品 |

### 3. 帮助信息

| # | 操作 | 预期结果 |
|---|------|----------|
| 3.1 | GM 输入 `//send`（无参数） | 显示 4 种格式帮助 + 数量上限提示 |

### 4. 基本发放

| # | 操作 | 预期结果 |
|---|------|----------|
| 4.1 | `//send100` | 背包获得物品 100 x1 |
| 4.2 | `//send100x5` | 背包获得物品 100 x5 |
| 4.3 | `//send100+10` | 背包获得物品 100 x1，强化 +10 |
| 4.4 | `//send100+10x3` | 背包获得物品 100 x3，强化 +10 |

### 5. 参数校验

| # | 操作 | 预期结果 |
|---|------|----------|
| 5.1 | `//sendabc`（非数字 ID） | 提示"物品代码无效"，不发物品 |
| 5.2 | `//send100x0`（数量 0） | 提示"数量无效"，不发物品 |
| 5.3 | `//send100x-1`（负数） | 提示"数量无效"，不发物品 |
| 5.4 | `//send100x999`（超 max_count） | 提示"数量超过上限"，不发物品 |
| 5.5 | `//send+10`（缺少 ID） | 提示"物品代码无效"，不发物品 |

### 6. 不存在物品的处理

| # | 操作 | 预期结果 |
|---|------|----------|
| 6.1 | `//send99999999`（不存在的物品 ID） | 提示"发放失败"，不发物品，记录日志 |

### 7. 日志记录

| # | 检查项 | 预期 |
|---|--------|------|
| 7.1 | 日志包含 GM 账号 | ✅ |
| 7.2 | 日志包含 GM 角色编号 | ✅ |
| 7.3 | 日志包含物品 ID | ✅ |
| 7.4 | 日志包含数量 | ✅ |
| 7.5 | 日志包含发放结果 | ✅ |
| 7.6 | 拒绝/失败也记录日志 | ✅ |

### 8. 不影响其他模块

| # | 操作 | 预期结果 |
|---|------|----------|
| 8.1 | `//view` | 正常查询物品 |
| 8.2 | `//myinfo` | 正常显示个人信息 |
| 8.3 | `//qd` | 正常签到（如已开启） |
| 8.4 | `//指令` | 正常显示菜单 |
| 8.5 | 其他未命中 GmInput | 正常透传 |
