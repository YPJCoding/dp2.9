# 安全规则

## 默认关闭

所有业务功能默认关闭。

## 高风险功能

以下能力属于高风险：

- 发物品
- 发点券 / 代币 / SP / TP / QP
- SQL update / delete
- shell 命令
- 清理背包
- 修改角色状态
- 内存 patch
- Frida hook
- 经济系统相关逻辑

高风险功能必须：

1. 默认关闭。
2. 独立 risk 开关。
3. GM-only 或明确权限控制。
4. 完整日志。
5. 测试服验证后才能启用。

## SQL 规则

禁止：

- 通用 SQL 执行器
- 用户输入直接拼 SQL
- 无白名单字段 update
- delete 语句默认开启

## JS/Frida 规则

禁止：

- 无注释真实地址
- 无重复 hook 防护
- 无配置开关的 hook
- 入口文件直接堆大量业务逻辑
